# 图书推荐系统

您好，这是我们“二元一次方程组”对于第一届慧科杯人工智能应用创新大赛提交的作品。含数据集+Flask+模型+模型代码等。

#### 如果您关心代码的运行，请看：

##### 开发&运行环境：

Python3.8

其他环境：

```requirements.txt```中为开发过程中所依赖的环境。

安装完环境之后，在项目根目录执行```python app.py```等待数秒即可启动成功。
